# About The Game
This is a VR game created on Unity 6.3 and tested on the Meta Quest 3.<br/>
After being chased out of a pizzeria, you decide to make a living by making food in a little subway train's kitchen. However, this kitchen is a little more mystical and mischievous than it seems. With the challenge of magical ingredients that like to play around, successfully serving up a dish here proves to be a challenge!
